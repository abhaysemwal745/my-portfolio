# Personal Portfolio Website

A responsive personal portfolio website built to showcase my frontend projects, skills, and hands-on learning. This project focuses on clean UI, responsive layouts, and simple interactive behavior using core web development technologies.

## Features
- Responsive layout optimized for different screen sizes
- Clean and minimal UI with smooth transitions
- Fixed navigation header for easy section access
- Project showcase section with interactive cards
- Skills section grouped into clear categories
- Contact section with quick email access
- Cross-browser compatible design

## Tech Stack
- HTML – Semantic structure and accessible markup  
- CSS – Flexbox, Grid, responsive design, animations, and transitions  
- JavaScript – Basic interactivity and UI behavior  
- Git & GitHub – Version control and project management  

## Project Structure
├── index.html
├── style.css
├── script.js
└── assets
